# منصة إدارة أعمال مع تتبع المبيعات والتسويق والعملاء
==========================

## ملخص المشروع
----------------

منصة إدارة أعمال مع تتبع المبيعات والتسويق والعملاء هي تطبيق إلكتروني يهدف إلى تسهيل إدارة الأعمال وتتبع المبيعات والتسويق والعملاء. يتيح للمستخدمين إدارة البيانات بشكل فعال وتحليلها لتحسين أداء الأعمال.

## هيكل المشروع
----------------

### مكونات المشروع

* `backend`: يحتوي على الكود المصدر للباكند
* `frontend`: يحتوي على الكود المصدر للفرونتند
* `database`: يحتوي على البيانات والاستعلامات

### هيكل الملفات


.
├── backend
│   ├── app.py
│   ├── config.py
│   ├── models.py
│   └── routes.py
├── frontend

│   ├── styles.css
│   └── script.js
├── database
│   ├── schema.sql
│   └── data.sql
└── docker-compose.yml


## تثبيت وتشغيل المشروع
-------------------------

### خطوات تشغيل المشروع

1. تأكد من وجود Docker و Docker Compose مثبتين على جهازك.
2. افتح терمينال أو командية وانتقل إلى مجلد المشروع.
3. استخدم الأمر التالي لتشغيل المشروع:
bash
docker-compose up

4. افتح متصفحك ومرر إلى `http://localhost:5000` لتشغيل التطبيق.

### تفاصيل تشغيل المشروع

* `docker-compose.yml`: يحتوي على تفاصيل تشغيل المشروع.
* `docker-compose up`: يبدأ تشغيل المشروع.

## مكونات المشروع
-----------------

### مكونات الباكند

* `app.py`: يحتوي على الكود المصدر للباكند.
* `config.py`: يحتوي على تفاصيل الإعدادات.
* `models.py`: يحتوي على تعريفات الموديلات.
* `routes.py`: يحتوي على تعريفات الطرق.

### مكونات الفرونتند

* `index.html`: يحتوي على HTML للصفحة الرئيسية.
* `styles.css`: يحتوي على CSS للتصميم.
* `script.js`: يحتوي على JavaScript للوظائف.

### مكونات قاعدة البيانات

* `schema.sql`: يحتوي على تعريفات الجداول.
* `data.sql`: يحتوي على البيانات.

### أدوار المشروع

* `admin`: يحتوي على صلاحيات إدارة المشروع.
* `user`: يحتوي على صلاحيات المستخدمين.
